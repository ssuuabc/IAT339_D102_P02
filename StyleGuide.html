Navigation Button

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">Default Button</a>

Rendered element:

Homepage

In-text link

CSS selector:

.in-text-link

Sampe code:

<a class="in-text-link" href="#">this is my in-text link</a>

Rendered element:

this is my in-text link

Submit/Reset Buttons

CSS selector:

.button-reset, .button-submit

Sample code:

<a class="button-reset" href="#">Reset</a>
<a class="button-submit" href="#">Submit</a>

Rendered element:

Reset Submit

Text Input Field and Label

CSS selector:

.input-field

Sample code:

<form class="input-field">
<label for="form-username">Username:</label>
<input type="text" id="form-username">
</form>
